Mikhmon on web server OpenWRT/LEDE

Kelanjutan project bulan November web server uhttpd pada OpenWRT/LEDE yg gagal eksekusi tugas php aplikasi Mikhmon. Ternayata aplikasi Mikhmon pada OpenWRT/LEDE membutuhkan php7-mod-gmp. Agar Mikhmon berkerja dengan baik pada OpenWRT/LEDE silahkan ikuti langkah berikut :

  1. Config uhttpd
  2. Install PHP
  3. Config PHP
  4. Upload mikhmon ke OpenWRT/LEDE
