Mikrotik Hotspot Monitor

Dengan Aplikasi Mikrotik Hotspot Monitor [mikhmon] usaha hotsppot RT/RW NET akan semakin mudah. Mikhmon dapat membuat user sesuai dengan lisensi pada Router Board Mikrotik yang digunakan.

Fitur

  1. Multi platform (Windows, GNU/Linux, Android)
  2. Menampilkan User Hotspot yang aktif dan masa aktifnya.
  3. Menambah, edit dan hapus User Profile.
  4. Menampilkan daftar User Hotspot berdasarkan User Profile.
  5. Menghapus User Hotspot berdasarkan Username.
  6. Generate Voucher.
    • Generate Kode Voucher.
    • Generate User Password.
    • Generate Custom User Password.
    • (Generate Kode Voucher/User Password maksimal 99 untuk sekali generate)
  7. Cetak Voucher. (Ukuran kertas A4 atau F4)
  8. Custom warna Voucher.
  9. Menghapus otomatis user hotspot expired.
  10. Tools :
    • Add Remove DNS Static untuk blok website.
    • Log Mikrotik
    • History

Fitur andalan mikhmon yaitu dapat menghapus user/voucher yang expired secara otomatis. Sehingga tidak direpotkan lagi dengan user yg sudah expired. Pemilik hotspot tinggal generate voucher --> cetak --> jual = $ :).

Panduan Install di Windows

mikhmon-standalone (untuk Windows 7+)

  • Dowload mikhmon-standalone.
  • Ekstak file mikhmon-standalone.zip.
  • Buka folder Mikrotik Hotspot Monitor, kemudian jalankan file mikhmon-server.bat.
  • Untuk pengguna anti virus Avast mikhmon-server.exe akan di scan beberapa saat, tunggu saja.
  • Buka di browser (Firefox atau Google Chrome terbaru) http://127.0.0.1:8080
  • Login aplikasi menggunakan username dan password default (admin|1234).
  • Pada laman Setup isikan IP, username dan password Mikrotik, jangan lupa simpan, cukup dengan menekan tombol Enter.
  • Tes koneksi ke Mikrotik, klik pada tombol <-->.
  • Jika muncul informasi tentang Mikrotik, berarti aplikasi siap digunakan.
  • Untuk selanjutnya login menggunakan username dan password Mikrotik.

Web Server

  • Dowload USBWebserver dan Mikrotik Hotspot Monitor.
  • Buat folder WebServer, ekstak file USBWebserver.zip ke folder tersebut.
  • Ekstrak file mikrotik-hotspot-monitor-master.zip
  • Cari folder mikhmon dalam folder mikrotik-hotspot-monitor-master yang sudah diekstrak sebelumnya, kemudian di-copy-kan ke folder root WebServer.
  • Jalankan usbwebserver.exe kemudian pilih bahasa, dan tunggu samapi indikator hijau muncul.
  • Buka di browser (Firefox atau Google Chrome terbaru) http://127.0.0.1:8080/mikhmon
  • Login aplikasi menggunakan username dan password default (admin|1234).
  • Pada laman Setup isikan IP, username dan password Mikrotik, jangan lupa simpan, cukup dengan menekan tombol Enter.
  • Tes koneksi ke Mikrotik, klik pada tombol <->.
  • Jika muncul informasi tentang Mikrotik, berarti aplikasi siap digunakan.
  • Untuk selanjutnya login menggunakan username dan password Mikrotik.

Install di Android

Web Server

  • Dowload Bit Web Server dan Mikrotik Hotspot Monitor.
  • Install Bit Web Server di Android.
  • Ekstrak file mikrotik-hotspot-monitor-master.zip
  • Cari folder mikhmon dalam folder mikrotik-hotspot-monitor-master yang sudah diekstrak sebelumnya, kemudian di-copy-kan ke folder www di memory internal Android.
  • Jalankan Bit Web Server, posisikan On pada bagian Web server.
  • Buka di browser (Firefox atau Google Chrome terbaru) http://127.0.0.1:8080/mikhmon
  • Login aplikasi menggunakan username dan password default (admin|1234).
  • Pada laman Setup isikan IP, username dan password Mikrotik, jangan lupa simpan, cukup dengan menekan tombol Enter.
  • Tes koneksi ke Mikrotik, klik pada tombol <->.
  • Jika muncul informasi tentang Mikrotik, berarti aplikasi siap digunakan.
  • Untuk selanjutnya login menggunakan username dan password Mikrotik.

Install di GNU/Linux (keluarga Debian dan Ubuntu)

Web Server PHP

  • Dowload Mikrotik Hotspot Monitor.
  • Ekstrak file mikrotik-hotspot-monitor-master.zip
  • Copy folder mikhmon ke direktori home ~/
  • sudo apt update
  • sudo apt install php
  • sudo chmod +x ~/mikhmon/install/mikhmon-server
  • sudo chmod +x ~/mikhmon/install/mikhmon.desktop
  • sudo cp ~/mikhmon/install/mikhmon-server /bin/
  • cp ~/mikhmon/install/mikhmon.desktop ~/
  • Jalankan mikhmon-server yang ada di folder home ~/
  • Buka di browser (Firefox atau Google Chrome terbaru) http://127.0.0.1:8080/mikhmon
  • Login aplikasi menggunakan username dan password default (admin|1234).
  • Pada laman Setup isikan IP, username dan password Mikrotik, jangan lupa simpan, cukup dengan menekan tombol Enter.
  • Tes koneksi ke Mikrotik, klik pada tombol <->.
  • Jika muncul informasi tentang Mikrotik, berarti aplikasi siap digunakan.
  • Untuk selanjutnya login menggunakan username dan password Mikrotik.
